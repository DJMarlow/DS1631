# `DS1631` library

Arduino library for communication with a DS1631

## Usage
- Download and copy the `DS1631` folder to your Arduino libraries folder (usually `C:\Users\<user_name>\Documents\Arduino\libraries`).
- Include the library in your Arduino code:
```
#include "DS1631.h"
```
